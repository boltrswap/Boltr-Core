# Boltr Core (Factory)

Documentation on BoltrSwap is available at [docs.boltrswap.com](https://docs.boltrswap.com/).

# Local Development

The following assumes the use of `node@>=10`.

## Install Dependencies

`yarn`

## Compile Contracts

`yarn compile`

## Run Tests

`yarn test`
